# Super Calculador de IMC - Brainstorming de Design

## Resposta 1: Futurismo Minimalista Sci-Fi (Probabilidade: 0.08)

**Movimento de Design:** Cyberpunk Minimalista + Glassmorphism

**Princípios Centrais:**
- Linhas limpas, geométricas e precisas
- Transparência e profundidade através de vidro frosted
- Contraste alto entre elementos
- Movimento sutil mas constante

**Filosofia de Cores:**
- Fundo: Preto profundo (#0a0e27) com gradiente sutil para azul escuro
- Primário: Ciano neon (#00d9ff) para elementos interativos
- Secundário: Magenta (#ff006e) para destaques
- Acentos: Verde neon (#39ff14) para estados de sucesso
- Vidro: Branco com 8% opacidade para cards

**Paradigma de Layout:**
- Grid assimétrico com cards flutuantes
- Painel central de cálculo com efeito de profundidade 3D
- Recomendações em cards deslizáveis horizontais
- Divisores diagonais com SVG animado

**Elementos Assinatura:**
- Linhas de brilho (glow lines) que animam ao redor de elementos
- Partículas flutuantes no fundo
- Efeito de scan horizontal que passa sobre o painel principal
- Bordas com gradiente neon

**Filosofia de Interação:**
- Hover: Elementos ganham brilho e se elevam
- Click: Pulso de energia que se expande
- Resultado: Animação de transição com efeito de "decodificação"

**Animações:**
- Entrada: Fade + slide de baixo com delay escalonado
- Hover: Scale 1.02 + glow intenso
- Cálculo: Animação de números contadores + efeito de glitch suave
- Resultado: Transição com efeito de cortina que revela recomendações

**Sistema Tipográfico:**
- Display: Orbitron (futurista, geométrico)
- Corpo: Inter (legível, moderno)
- Hierarquia: Orbitron 32px para título, Inter 16px para corpo

## Resposta 2: Premium Wellness Moderno (Probabilidade: 0.07)

**Movimento de Design:** Luxury Minimalism + Wellness Aesthetic

**Princípios Centrais:**
- Espaço generoso e respiração visual
- Cores naturais com acentos sofisticados
- Tipografia elegante e hierarquizada
- Transições suaves e confortáveis

**Filosofia de Cores:**
- Fundo: Branco quente (#faf9f7) com padrão de ruído subtil
- Primário: Azul profundo (#1a3a52) para confiança
- Secundário: Coral suave (#f4a582) para energia positiva
- Acentos: Verde menta (#a8d5ba) para bem-estar
- Destaques: Ouro suave (#d4af37) para premium

**Paradigma de Layout:**
- Layout assimétrico com coluna lateral
- Painel central com sombra suave e espaçamento generoso
- Recomendações em cards com ícones ilustrados
- Tipografia grande e respirada

**Elementos Assinatura:**
- Ícones ilustrados em estilo line-art
- Sombras suaves (blur 20px, opacity 0.1)
- Padrão de gradiente sutil no fundo
- Decorações geométricas minimalistas (círculos, linhas)

**Filosofia de Interação:**
- Hover: Elevação suave com sombra expandida
- Click: Feedback tátil com mudança de cor
- Resultado: Transição elegante com reveal de conteúdo

**Animações:**
- Entrada: Fade + scale suave
- Hover: Elevação 4px + sombra expandida
- Cálculo: Números animados com easing suave
- Resultado: Fade in progressivo das recomendações

**Sistema Tipográfico:**
- Display: Poppins Bold (elegante, moderno)
- Corpo: Lato (legível, quente)
- Hierarquia: Poppins 36px para título, Lato 16px para corpo

## Resposta 3: Energia Dinâmica Gradiente (Probabilidade: 0.06)

**Movimento de Design:** Kinetic Typography + Motion Design

**Princípios Centrais:**
- Movimento como elemento principal de design
- Gradientes vibrantes e dinâmicos
- Formas orgânicas e fluidas
- Energia e motivação em cada interação

**Filosofia de Cores:**
- Fundo: Gradiente animado de roxo (#6d28d9) a rosa (#ec4899)
- Primário: Amarelo vibrante (#fbbf24) para ação
- Secundário: Laranja (#f97316) para destaque
- Acentos: Branco (#ffffff) para contraste
- Overlay: Vidro com gradiente semi-transparente

**Paradigma de Layout:**
- Layout fluido com cards em cascata
- Painel central com efeito de flutuação
- Recomendações em cards com fundo gradiente individual
- Divisores com formas orgânicas (wave, blob)

**Elementos Assinatura:**
- Formas blob animadas no fundo
- Gradientes que mudam de cor ao hover
- Ícones com animação de rotação suave
- Linhas de movimento que guiam o olhar

**Filosofia de Interação:**
- Hover: Rotação suave + mudança de gradiente
- Click: Pulso de escala + mudança de cor
- Resultado: Animação explosiva com confete de cores

**Animações:**
- Entrada: Bounce suave + fade
- Hover: Rotação 5° + scale 1.05 + mudança de gradiente
- Cálculo: Animação de contagem com easing bounce
- Resultado: Animação de confete + transição de cores

**Sistema Tipográfico:**
- Display: Montserrat Bold (dinâmico, moderno)
- Corpo: Open Sans (legível, amigável)
- Hierarquia: Montserrat 40px para título, Open Sans 16px para corpo

---

## Decisão Final

Será implementado o **Design 1: Futurismo Minimalista Sci-Fi** porque:
- Transmite modernidade, sofisticação e energia
- Diferencia-se de designs genéricos
- Suporta animações complexas e visualmente impressionantes
- Cria uma experiência premium e memorável
- Alinha-se perfeitamente com o conceito de "super calculador"
